# Car-Specification-Dashboard
This project is an interactive Excel dashboard created to analyze car specifications using an Indian car dataset (Kaggle).
The dashboard focuses on comparing car performance, pricing, manufacturers, and body types through interactive visualizations.
Unlike traditional car dashboards that focus on sales, this project analyzes specifications to provide insights into the available models.

### Objectives
The dashboard answers questions such as:

Which manufacturers offer the highest average priced vehicles?
Which cars have the highest engine power?
How are cars distributed across various performance categories?
How does engine power relate to vehicle price?
How do specifications change when filters are applied?

### Dataset
The dataset contains information about passenger vehicles sold in India.

### Data Cleaning
The following preprocessing steps were performed:

Removed duplicate records
Corrected inconsistent specification values
Standardized performance categories
Validated numerical fields
Handled missing values where appropriate

### Key Insights

Luxury manufacturers command significantly higher average prices.
Higher engine power generally corresponds to higher prices.
SUVs are among the most common body types in the dataset.
Most vehicles fall into the Standard and Powerful performance categories.
